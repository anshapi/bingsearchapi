🔍 AnshAPI - Bing Search API

This API provides web and image results from Bing search.

🔧 How to use

Endpoint: /api/search?search=QUERY&state=web|image

  • search: your search term
  • state: web or image

🧪 Examples

  • Image Search: dog
  • Web Search: elon musk
API by @anshapi on Telegram • Free to use 🔗
